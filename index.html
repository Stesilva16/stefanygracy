Habilidades

Técnico

Dados

  • SQL (Structured Query Language) - é uma linguagem de consulta usada para manipular dados em bancos de dados relacionais, como MySQL.
  • Python - é uma linguagem de programação amplamente utilizada na área de ciência de dados, com muitas bibliotecas e ferramentas para processamento de dados, aprendizado de máquina e visualização de dados.
  • Spark - é uma plataforma de processamento de dados em cluster que fornece uma API em Python, Scala ou Java.
  • PowerBI e Looker Studio - são ferramentas de visualização de dados que permitem criar dashboards interativos e relatórios para ajudar a explorar e compreender os dados.
  • Data Warehouse é um sistema de armazenamento de dados que permite a integração, consulta e análise de grandes quantidades de dados de diversas fontes. É uma tecnologia de suporte para a tomada de decisões, auxiliando na gestão de negócios.
  • Data lake é um sistema de armazenamento de dados não estruturados, semi-estruturados e estruturados, onde é possível integrar dados de diferentes fontes e aplicar processos de análise em grande escala. É uma solução de Big Data que permite a gestão de dados não estruturados e permite a criação de insights e análises de negócios mais robustos.
  • ETL é a sigla em inglês para Extraction, Transformation, and Loading. É um processo de extração de dados de fontes diferentes, transformação para atender às necessidades de negócios específicos e carregamento em um sistema centralizado, geralmente um data warehouse ou um banco de dados.

    • Front-End

  • HTML é uma linguagem de marcação usada para criar páginas da web.
    - Tags: são elementos utilizados para marcar o conteúdo de uma página, como títulos, parágrafos, imagens, links, etc.
    - Atributos: são valores adicionais que podem ser incluídos nas tags, para modificar seu comportamento ou apresentação.
    - Hierarquia: as tags HTML são organizadas em uma estrutura hierárquica, onde as tags mais externas envolvem as internas, criando uma estrutura de conteúdo.
    -Semântica: o HTML permite que os desenvolvedores adicionem significado ao conteúdo da página, de forma que ele possa ser entendido por programas e dispositivos.
  • CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML. É usado para controlar a aparência e formatação de páginas da web, incluindo cores, fontes, layouts, margens, tamanhos de elementos, entre outros aspectos. Além disso, o CSS permite a separação do conteúdo e da apresentação da página, permitindo que designers e desenvolvedores trabalhem de forma mais eficiente e mantenham uma aparência consistente em todo o site.
  • JavaScript é uma linguagem de programação de alto nível, dinâmica e interpretada, que é amplamente utilizada no desenvolvimento web. Ele é usado para adicionar interatividade e recursos avançados às páginas da web, como formulários, animações, gráficos e jogos.
